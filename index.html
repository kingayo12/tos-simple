Gate Operations Overview

Gate In Containers entering the terminal/yard. This could be imported cargo coming in, or empty containers returned from ports into the yard.
Gate Out Containers leaving the terminal/yard. This includes export releases, empty returns, container sales, or imports being delivered out.

Gate In – Operation Types

Type What It Means What Happens at Gate Key Details Captured
Import Loaded container arriving into the yard from port or vessel discharge. Truck is checked in, container is inspected, documents are verified. Agent, Driver, Truck No, Container No, Size, Shipping Line, Receiver
Empty Empty container coming into the yard from port. Truck enters yard, container condition is confirmed. Port, Container No, Size, Driver, Truck No, Shipping Line

Gate Out – Operation Types

Type What It Means What Happens at Gate Key Details Captured
Import Loaded container leaving yard for final delivery. Release is confirmed, container is verified before exit. Agent, Driver, Truck No, Container No, Size, Shipping Line, Release Officer
Empty Empty container leaving yard back to port. Container is checked out, port destination is confirmed. Port Destination, Container No, Size, Driver, Truck No, Shipping Line
Export Release Export container leaving yard for vessel loading. Booking is validated, exporter details are confirmed. Booking No, Exporter, Agent, Driver, Truck No, Container No, Size
Empty Return Empty container returned to shipping line. Container is released and condition logged. Container No, Size, Agent, Driver, Truck No, Shipping Line
Container Sales Sold container leaving the yard permanently. Buyer identity is confirmed, container ownership is transferred. Agent, Company, Container No, Size, Driver, ID Card, Shipping Line

Common Events at the Gate

Document Verification Agent papers, booking numbers, release documents are checked.
Container Inspection Physical condition, container number, seal, and size are verified.
Driver & Truck Validation Driver identity and truck number are confirmed before entry or exit.
