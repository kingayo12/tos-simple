Gate Operations Overview

Gate InContainers entering the terminal/yard — imported cargo arriving, or empty containers returned from ports.
Gate OutContainers leaving the terminal/yard — export releases, empty returns, container sales, or import deliveries.

Gate In — Operation Types

Type What It Means What Happens at Gate Key Details
Import Loaded container arriving into yard from port/vessel. Truck checked in, container inspected, documents verified. Agent, Driver, Truck, Container, Size, Shipping, Receiver
Empty Empty container arriving from port into yard. Truck enters yard, container condition confirmed. Port, Container, Size, Driver, Truck, Shipping

Gate Out — Operation Types

Type What It Means What Happens at Gate Key Details
Import Loaded container leaving yard for final delivery. Release confirmed, container verified before exit. Agent, Driver, Truck, Container, Shipping, Release Officer
Empty Empty container leaving yard back to port. Container checked out, port destination confirmed. Port Dest., Container, Size, Driver, Truck, Shipping
Export Export container leaving for vessel loading. Booking validated, exporter details confirmed. Booking, Exporter, Agent, Driver, Truck, Container
Empty Return Empty container returned to shipping line. Container released and condition logged. Container, Size, Agent, Driver, Truck, Shipping Co.
Sales Sold container leaving yard permanently. Buyer identity confirmed, ownership transferred. Agent, Company, Container, Size, Driver ID, Shipping
